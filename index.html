Open this viewer over HTTP on your PC

Twitch does not allow embedded video when the page is opened as a file (file:// in the address bar). That is required by Twitch, not a limitation of this project.

To run it locally: double-click View Twitch Viewer.bat in this folder. It starts a small server on your machine only and opens http://127.0.0.1:3000 in your browser.

The first time, install Node.js if Windows reports that node was not found, then run the batch file again.
